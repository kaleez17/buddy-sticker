# 🎨 Buddy Sticker Shop — Demo

A premium React storefront for Buddy Sticker Shop, Adyar, Chennai.

## Features
- Homepage with hero, stats & featured products
- Shop page with category filters
- Product detail page with size/colour/qty selection
- Cart sidebar with live totals
- Checkout flow
- About page with real shop info

## Tech Stack
- React 18 + Vite
- Pure CSS (no external UI library)
- Font: Cormorant Garamond + DM Sans + DM Mono (Google Fonts)

---

## 🚀 Host on Vercel (Recommended — Free)

### 1. Install dependencies & run locally
```bash
npm install
npm run dev
```

### 2. Deploy to Vercel
```bash
npm install -g vercel
vercel
```
Follow the prompts. You'll get a live link like:
`https://buddy-sticker-shop.vercel.app`

---

## 🌐 Host on Netlify (Drag & Drop — Easiest)

```bash
npm install
npm run build
```

Then go to **https://app.netlify.com/drop** and drag the `dist/` folder.
Live in 30 seconds. No account needed.
